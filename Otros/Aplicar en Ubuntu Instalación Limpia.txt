# NO APLICAR NO APLICAR NO APLICAR - SOLO FUNCIONA CON ARCHLINUX
# Pantalla parpadeo / Forzo de cursor dañado (iGPU):
Panel Self Refresh (PSR), una función de ahorro de energía utilizada por Intel iGPUs es conocida por causar parpadeo en algunos casos FS-49628 FS-49371 FS-5605. Una solución temporal es desactivar esta característica utilizando el parámetro del núcleo i915.enable_psr=0
# Esto deshabilitará una función de ahorro de energía que principalmente está causando este problema.
# Grub bootloadloader:
# Para habilitarlo de la manera más fácil, es configurarlo en la comida, añadiéndolo a la lista GRUB-CMDLINE-LINUX-DEFAULT:
sudo nano /etc/default/grub
GRUB_CMDLINE_LINUX_DEFAULT="quiet loglevel=3 nowatchdog i915.enable_psr=0"
# Guarde el archivo [Ctrl-x] y reconstruir grub.cfg:
sudo grub-mkconfig -o /boot/grub/grub.cfg
